# Tuya嵌入式多媒体SDK


## 介绍
Tuya嵌入式多媒体SDK是在涂鸦IoTOS体系中针对具备音视频能力的嵌入式设备，如网络摄像机（IPC）、录像机（NVR/DVR）、可视门铃（doorbell）、摄像头灯（Floodloght）等推出的软件开发包。提供标准C语言静态库、头文件以及对应的说明文档，覆盖Linux、LiteOS以及部分特定的RTOS系统。

## 下载中心
